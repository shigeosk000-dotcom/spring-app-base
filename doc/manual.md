# ToDoアプリ 利用マニュアル

## アプリ起動編
1. **前提**: Java 17 がインストールされていることを確認します。
2. リポジトリ直下で `./mvnw spring-boot:run`（Windows なら `cmd /c "set MAVEN_USER_HOME=%CD%\.m2 && mvnw.cmd spring-boot:run"`）を実行してアプリを起動します。ローカル環境で `MAVEN_USER_HOME` を書き込み可能なディレクトリ（例: リポジトリ直下の `.m2`）に設定すると、ローカルリポジトリが確実に作成されます。
3. 起動後はブラウザで `http://localhost:8080/todos` にアクセスすると、ToDo 一覧画面が表示されます。
4. 管理者が直接 DB を確認したい場合は `http://localhost:8080/h2-console` にアクセスし、JDBC URL に `jdbc:h2:mem:todo`、ユーザー名 `sa`、パスワード空欄でログインできます。

## 管理者編
- 初期データは `src/main/resources/data.sql` で定義されているため、起動時に自動でロードされます。
- H2 コンソールから `todo` テーブルを開き、レコードを直接閲覧・編集・削除できます。検索条件を設定して特定タスクだけを表示したり、`done` 列を `TRUE` にして完了状態を更新できます。
- アプリ UI では、各タスクの「編集」リンクから詳細情報（タイトル、説明、期日、優先度、完了状態）を更新できます。完了状態はチェックボックスでも切り替え可能です。
- メッセージは画面上部に表示され、操作の成否（追加・更新・削除）がフィードバックされます。

## 利用者編
1. 画面上部の一覧で現在のタスクを確認します。完了済みのタスクはグレーアウトし、行全体に取り消し線が入ります。
2. 「完了にする」ボタンで作業中のタスクを完了済みに、完了済みタスクは「再開する」で未完了に戻せます。
3. 「編集」リンクでタスクの内容（タイトル、説明、期日、優先度、完了フラグ）を修正できます。編集画面では変更後に「更新する」で保存し、「キャンセル」で一覧に戻ります。
4. フォームにタイトルを入力し、必要なら説明・期日・優先度・完了フラグを設定して「登録する」を押すと新しいタスクを追加できます。主キーは自動で採番されます。
5. 不要になったタスクは「削除」ボタンで一覧から取り除けます。
## 追加仕様の確認

- `doc/fsd.md` を参照し、ユーザー管理の画面構成（一覧／登録／編集／状態切替／詳細確認）とタスクへのユーザー紐づけを実装しています。
- `schema.sql`/`data.sql` は user_list と todo を再定義し、ON DELETE CASCADE・日本語メッセージ付きステータスも整備しました。
- `/todos/user/*` テンプレートと `/todos/login` の簡易画面を追加しました。
